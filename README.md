# strategem-hero

Browser trainer for **Helldivers 2** stratagem codes (keyboard, swipes, local assets).

**Live site (after you enable Pages):** [https://volkula.github.io/strategem-hero/](https://volkula.github.io/strategem-hero/)

## GitHub Pages (one-time)

1. Repo **Settings** ? **Pages**
2. **Build and deployment** ? Source: **GitHub Actions**
3. Push to `main` — workflow **Deploy GitHub Pages** publishes the site

## Run locally

Open `index.html` in a browser or serve the folder with any static server.
